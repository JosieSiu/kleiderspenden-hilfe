# kleiderspenden-hilfe
Öffentliches Webportal zur Registrierung von Kleiderspenden für
verschiedene Krisenregionen.

Funktionen
Registrierung von Kleiderspenden
Auswahl zwischen Übergabe vor Ort und Abholung

PLZ-Prüfung bei Abholung
Bestätigungsseite mit allen Daten
Responsive Design für Desktop, Tablet und Smartphone

Technologien
HTML5
CSS3 (Responsive Layout)
JavaScript (Validierung & Logik)

Nutzung
Die Anwendung kann lokal im Browser geöffnet oder über GitHub Pages
bereitgestellt werden.
